Press Spacebar to Start

Rules

  • 1. The game consists of multiple levels. The objective is to repeat the sequence of buttons played in each level.
  • 2. Each level has a time limit, depended of game difficulty. You must complete the sequence before the timer runs out.
  • 3. You are allowed to use up to 3 hints during each level. When you click the "Hint" button, the next button in the sequence will be shown.
  • 4. The sequence of buttons grows longer with each new level.
